Detailed Description

Das Struct Point beschreibt einen Punkt durch zwei Variablen x, y, einem Pointer auf den parent Knoten, einem Rang, um eine union-find Struktur auf diesem zu definieren, und einer Color, um die Knoten während des Traversierens der face-cycles, auf denen Punkte liegen, zu färben.
