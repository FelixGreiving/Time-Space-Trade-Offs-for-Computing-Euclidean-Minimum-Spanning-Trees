Detailed Description

Die Klasse EMST dient zur Berechnung und Ausgabe eines EMSTs zu einer Eingabepunktmenge. Im Konstruktor wird dabei der Algorithmus gemäß der Idee des Time Space Trade-Offs von Banyassadi implementiert, der auf O(s) Speicherzellen und in einer Laufzeit von O(n³/s²*log(s)) den EMST berechnet. Zudem werden die Delaunay Triangulation, der Relative Nachbarschaftsgraph, das S-Netz, der Hilfsgraph G für den jeweiligen partiellen Relativen Nachbarschaftsgraphen RNG_i und dessen face-cycles in eine Datei geschrieben.
